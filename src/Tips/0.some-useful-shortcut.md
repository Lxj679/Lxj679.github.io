## 1. windows 常用快捷键

- **复制**

`ctrl + c`

- **粘贴**

`ctrl + v`

- **剪切**

`ctrl + x`

- **截图**

`windows + shift + s`

- **全选**

`ctrl + a`

- **打开运行对话框**

`windows + r`





## 2. pycharm

- **放大（缩小）文本**

`ctr + 鼠标滚轮上（下）滑动`

- **将文本改为批注**

`ctrl  + /`

- **快速复制上一行**

`ctrl + d`



## 3. ConEmu

- **运行本地**

`pnpm run docs:dev`

- **退出本地**

`ctr + c`

- **运行本地网址**

`ctr + 点击网址链接`

- **放大（缩小）命令行文本**

`ctr + 鼠标滚轮上（下）滑动`
