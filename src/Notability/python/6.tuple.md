## 1. 创建元组

- 使用小括号创建；
- 里面元素用英文逗号隔开；

```python
tup = ('a', 'b', 'c')
print(tup, type(tup))

# output
('a', 'b', 'c') <class 'tuple'>
```

## 2. 列表和元组的对比

![image-20240415110337842](./6.tuple.assets/image-20240415110337842.png)

列表和元组的区别：

- 列表中：元素用方括号`[]`包裹；元组中则是`。
- 列表中的元素可以被修改、添加、删除，即列表是可变的数据类型，元组是不可变的数据类型。

## 3. 元组是不可变的

## 4. 元组的取值和分片操作

```python
tup  = (2, 'lilei', 19)
print(tup) # (2, 'lilei', 19)
print(tup[0]) # 2
print(tup[::2]) # (2, 19)
print(tup[::-1]) # (19, 'lilei', 2)

# output
(2, 'lilei', 19)
2
(2, 19)
(19, 'lilei', 2)
```

::: tips Question

基于上面的元组，提取得到：`('lilei', 2)`

```python
tup = (2, 'lilei', 19)
```
