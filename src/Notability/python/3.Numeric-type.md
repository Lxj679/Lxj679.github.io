## 1. 数字型的特点

```python
In [1]: 1 + 1
Out[1]: 2

In [2]: 2 - 1
Out[2]: 1

In [3]: 1 + 1.0
Out[3]: 2.0

In [4]: 2 * 2
Out[4]: 4

In [5]: 2 * 2.0
Out[5]: 4.0

In [6]: 9 / 3
Out[6]: 3.0
```

::: tips 

- 如果其中有一个浮点数，结果就会得到浮点数（优先级最高）
- 除法涉及精度问题，所以最后的结果类型：就是浮点数

:::

## 2. 算术运算符

算术运算符：用于算术计算。

| 运算符 | 描述                                   | 例子        |
| ------ | -------------------------------------- | ----------- |
| `+`    | 加法运算符                             | 1 + 1 = 2   |
| `-`    | 减法运算符                             | 2 – 1 = 1   |
| `*`    | 乘法运算符                             | 2 * 3 = 6   |
| `/`    | 除法运算符                             | 9 / 3 = 3.0 |
| `**`   | 指数运算符                             | 2 ** 3 = 8  |
| `%`    | 取余运算符，计算余数                   | 9           |
| `//`   | 除法取整运算符，计算商并去除其小数部分 |             |
